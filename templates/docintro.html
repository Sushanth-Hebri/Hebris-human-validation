Hebris Human Image Verifier API Documentation


Hebris V1.0

Endpoint: https://hebris-human-validation.onrender.com/predict

Detects whether an uploaded image is valid image of human

Request:

POST /predict

Form Data:

<form action="https://hebris-human-validation.onrender.com/predict" method="post" enctype="multipart/form-data">
  <input type="file" name="image" accept="image/*" required>
  <button type="submit">Detect</button>
</form>

Response:

{ "predicted_class": "human" }

Note: Replace /detect with your API endpoint.

Use Cases

1.User Profile Picture Verification

Validate user-uploaded profile pictures to ensure they contain human faces, improving the authenticity of user accounts.

// Example code snippet for user profile picture verification const fileInput = document.getElementById('profileImageInput'); const formData = new FormData(); formData.append('image', fileInput.files[0]); fetch('/predict', { method: 'POST', body: formData }) .then(response => response.json()) .then(data => console.log(data));

2.Detect Human Presence in Security Cameras

This API can be used to analyze images captured by security cameras and determine whether there is a human presence in the scene.

// Example code snippet for security camera integration const imageUrl = 'https://example.com/security-camera/image.jpg'; const formData = new FormData(); formData.append('image', imageUrl); fetch('/predict', { method: 'POST', body: formData }) .then(response => response.json()) .then(data => console.log(data));

3.Content Moderation in Social Media

Automatically filter out or flag images that contain human subjects for moderation purposes, ensuring compliance with community guidelines and policies.

Categorize product images based on the presence of humans, such as apparel being modeled by humans versus product images without human models, for better organization and searchability.

Currently, this API is provided as a free version (V1.0).


This API is build using machine learning model (Keras) with a training dataset from Kaggle

Further model enhancement has scope and still open to further research, response time is little more when compared with network speed as it is hosted in a free tier on render
